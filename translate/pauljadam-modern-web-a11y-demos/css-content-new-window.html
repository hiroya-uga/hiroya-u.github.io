新しいウィンドウへのリンクの表現

新しいウィンドウへのリンクの表現に、BootstrapのCSSとFont Awesome Icons、JavaScriptによるtitle属性付与を利用します。

CSSで出力されたBootstrapやFont Awesome Frameworksの別窓アイコンと、別窓で開くことを知らせる説明テキストをtitle属性にJavaScriptで設定する

14/7/21 更新: このデモでは、title属性が代替テキストとして利用されているため、VoiceOverのアイコンフォントを抑制するために「CSSによって出力されたコンテンツのためのCSS4 Alt text」を追加しました。title属性ではなく、altプロパティが利用されている新しいDEMOをご覧ください。
